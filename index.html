Встановлення Windows

Windows є найпоширенішою операційною системою на ПК. У цьому розділі розглянемо детально, як правильно підготувати комп’ютер, створити завантажувальну флешку та виконати чисту установку або оновлення системи. Ці інструкції підходять для Windows 10 та 11, включаючи 32- та 64-бітні версії.

1. Вимоги до системи

Перед установкою переконайтеся, що ваш комп’ютер відповідає мінімальним вимогам:

  • Процесор: 1 ГГц або швидший (2 ядра для 64-bit)
  • Оперативна пам'ять: мінімум 2 ГБ (4 ГБ для 64-bit)
  • Вільне місце на диску: мінімум 20 ГБ
  • Відеокарта: сумісна з DirectX 9 або новіша
  • Підключення до інтернету для завантаження оновлень

2. Підготовка до установки

  1. Завантажте офіційний ISO образ з сайту Microsoft.
  2. Створіть завантажувальну флешку за допомогою Rufus або Ventoy. Для Rufus оберіть файлову систему NTFS та GPT для UEFI, або MBR для старих BIOS.
  3. Зробіть резервну копію всіх важливих файлів на зовнішньому диску або хмарному сховищі.
  4. Переконайтеся, що ваш комп’ютер завантажується з USB (змінити порядок завантаження в BIOS/UEFI).

3. Покрокова інструкція установки

  1. Вставте завантажувальну флешку та перезавантажте ПК.
  2. Увійдіть до BIOS/UEFI та оберіть флешку як перший завантажувальний пристрій.
  3. Після старту інсталятора оберіть мову, формат часу і розкладку клавіатури.
  4. Натисніть “Встановити зараз” та введіть ключ продукту, якщо є.
  5. Виберіть тип установки: “Оновлення” або “Чиста установка”. Для чистої установки оберіть розділ диска та форматування.
  6. Дочекайтеся завершення інсталяції, комп’ютер перезавантажиться кілька разів.
  7. Налаштуйте обліковий запис користувача та параметри конфіденційності.

4. Поради та фішки

  • Встановіть всі оновлення Windows одразу після першого запуску.
  • Використовуйте офіційні драйвери від виробника материнської плати та відеокарти.
  • Для dual-boot спершу ставте Windows, потім Linux, щоб уникнути перезапису завантажувача.
  • Створіть образ системи після чистої установки для швидкого відновлення у разі проблем.
  • Регулярно робіть резервне копіювання важливих файлів.

5. Поширені проблеми та рішення

  • “No bootable device” – перевірте порядок завантаження у BIOS/UEFI.
  • Флешка не визначається – перевірте формат і файлову систему USB.
  • Помилка активації – переконайтеся, що ключ продукту дійсний для вашої версії Windows.
  • Проблеми з драйверами – завантажуйте останні версії з офіційних сайтів виробників.

6. Додаткові ресурси

  • Rufus для створення завантажувальної флешки
  • Підтримка Microsoft

Встановлення Linux

Linux є відкритою операційною системою, яка має багато дистрибутивів для різних цілей. У цьому розділі розглянемо детальніше Ubuntu, Linux Mint і Fedora.

1. Вимоги до системи

  • Процесор: 1 ГГц або швидший
  • Оперативна пам’ять: 1–2 ГБ (4 ГБ для комфортної роботи з графічним середовищем)
  • Вільне місце на диску: мінімум 10 ГБ для установки базової системи
  • Підключення до інтернету для оновлень

2. Підготовка

  1. Завантажте ISO потрібного дистрибутива (Ubuntu, Mint, Fedora).
  2. Створіть завантажувальну флешку за допомогою Etcher або Rufus.
  3. Зробіть резервну копію важливих файлів.
  4. Якщо плануєте dual-boot, переконайтеся, що у вас є достатньо вільного місця на диску.

3. Покрокова інструкція установки

  1. Вставте флешку та перезавантажте комп’ютер, обравши флешку як перший пристрій завантаження у BIOS/UEFI.
  2. Запустіть Live-режим, щоб перевірити сумісність обладнання.
  3. Натисніть “Install” та оберіть мову і регіон.
  4. Виберіть тип установки: “Стерти диск” або “Встановити поруч із Windows” (dual-boot).
  5. Налаштуйте розділи, якщо потрібно: root, swap, home.
  6. Введіть ім’я користувача та пароль.
  7. Дочекайтеся завершення інсталяції та перезавантажте комп’ютер.

4. Поради та фішки

  • Оновіть систему: sudo apt update && sudo apt upgrade
  • Встановіть необхідне ПЗ через пакетний менеджер (apt, dnf, pacman).
  • Створіть резервну копію домашньої папки перед великими оновленнями.
  • Використовуйте Live-режим для перевірки обладнання перед установкою.
  • Навчіться користуватися терміналом, це значно спрощує налаштування системи.

5. Поширені проблеми та рішення

  • Система не завантажується – перевірте режим UEFI/Legacy у BIOS.
  • Wi-Fi не працює – встановіть пропрієтарні драйвери.
  • Помилки при розділенні диска – перевірте таблицю розділів (GPT/MBR).
  • Проблеми з оновленнями – перевірте джерела пакетів та підключення до інтернету.

6. Додаткові ресурси

  • Офіційний сайт Ubuntu
  • Офіційний сайт Linux Mint
  • Офіційний сайт Fedora

Інші операційні системи

Окрім Windows та Linux існують інші операційні системи, які використовуються для специфічних завдань або для навчання. Нижче наведено приклади та базові інструкції.

1. FreeBSD

  • Скачайте ISO з офіційного сайту.
  • Створіть завантажувальну флешку.
  • Перезавантажте ПК та оберіть флешку у BIOS/UEFI.
  • Слідуйте покроковому інсталятору FreeBSD для створення розділів та налаштування користувача.

2. ReactOS

  • ReactOS – експериментальна ОС з сумісністю Windows.
  • Завантажте ISO з офіційного сайту.
  • Створіть флешку та встановіть на окремий розділ або віртуальну машину.

3. macOS (Hackintosh)

  • Встановлення macOS на ПК можливе лише на сумісному обладнанні.
  • Використовуйте інструкції Hackintosh та спеціальні завантажувачі (OpenCore, Clover).
  • Обов’язково створюйте резервну копію даних перед установкою.

4. Поради та фішки

  • Перед встановленням перевіряйте сумісність обладнання.
  • Використовуйте віртуальні машини для тестування ОС без ризику для основної системи.
  • Читайте офіційну документацію кожної ОС для уникнення помилок.

5. Додаткові ресурси

  • FreeBSD
  • ReactOS

FAQ – Часті питання

  • Чи можна встановити Linux на комп’ютер із Windows 11? – Так, можна створити dual-boot або використовувати віртуальну машину.
  • Як відновити систему після невдалої установки? – Використовуйте резервні копії або створені образи диска.
  • Що робити, якщо система не бачить флешку? – Перевірте формат USB, порт та режим BIOS/UEFI.
  • Чи потрібно оновлювати драйвери після установки ОС? – Так, завжди використовуйте останні офіційні драйвери.

Dual-boot: Як зрозуміти вікно вибору ОС

Багато користувачів встановлюють Linux поруч із Windows (dual-boot) і при запуску комп’ютера бачать меню з вибором ОС. Це — нормальне явище. Воно відображається завантажувачем, зазвичай GRUB у Linux. У цьому вікні можна обрати, яку систему запускати.

Що бачить користувач:

  • Список встановлених операційних систем (наприклад, Windows 11 та Ubuntu).
  • Першу ОС у списку за замовчуванням (зазвичай Linux, якщо GRUB встановлено з Linux).
  • Таймер автоматичного запуску ОС за замовчуванням, якщо нічого не обрати.

Як користуватися меню вибору ОС:

  1. За допомогою стрілок ↑ та ↓ оберіть систему, яку хочете запустити.
  2. Натисніть Enter для запуску обраної ОС.
  3. Якщо не обрати нічого, система за замовчуванням завантажиться після закінчення таймеру.

Поради для новачків:

  • Не лякайтеся: це стандартне меню, воно не шкодить жодній системі.
  • Якщо хочете змінити ОС за замовчуванням або порядок списку, це можна зробити через налаштування GRUB у Linux.
  • Для більшої безпеки завжди створюйте резервні копії перед встановленням dual-boot.
  • Не вимикайте комп’ютер під час відображення завантажувача — просто дочекайтеся вибору ОС або таймеру.

  • GRUB у Arch Linux

BIOS/UEFI: Як розібратися та налаштувати

BIOS (Basic Input/Output System) або UEFI (Unified Extensible Firmware Interface) — це базова прошивка комп’ютера, яка запускається перед операційною системою. Через BIOS/UEFI можна перевірити апаратне забезпечення, налаштувати порядок завантаження, включити або вимкнути компоненти та підготувати ПК до установки ОС.

Навіщо заходити в BIOS/UEFI:

  • Змінити порядок завантаження (USB, HDD, SSD) для установки нової ОС.
  • Увімкнути або вимкнути Secure Boot (для Linux може знадобитися відключити).
  • Перевірити, чи система розпізнає оперативну пам’ять та диски.
  • Управляти функціями енергозбереження та апаратними налаштуваннями.

Як увійти в BIOS/UEFI:

  1. Перезавантажте комп’ютер.
  2. Під час старту натискайте клавішу, що відповідає за вхід до BIOS/UEFI (зазвичай Delete, F2, F10, Esc — див. документацію вашого ПК або материнської плати).
  3. Зачекайте, поки завантажиться екран налаштувань BIOS/UEFI.

Що можна налаштувати для встановлення ОС:

  • Порядок завантаження: встановіть USB або DVD як перший пристрій для завантаження інсталятора.
  • Secure Boot: деякі Linux-дистрибутиви не завантажуються при включеному Secure Boot — вимкніть його при необхідності.
  • UEFI/Legacy режим: вибір режиму впливає на сумісність з дисковими таблицями GPT/MBR.
  • Перевірка апаратного забезпечення: перевірте RAM, HDD/SSD, процесор, щоб уникнути проблем під час установки.
  • Збереження та вихід: після внесення змін обов’язково збережіть налаштування та перезавантажте ПК.

Поради для новачків:

  • Не змінюйте налаштування, про які ви не знаєте, щоб не пошкодити роботу ПК.
  • Перед зміною порядку завантаження або Secure Boot зробіть резервну копію даних.
  • Якщо після змін комп’ютер не завантажується, можна скинути BIOS/UEFI до заводських налаштувань.
  • Більшість сучасних материнських плат мають зручний графічний інтерфейс UEFI, де легко орієнтуватися новачкам.
